论文方向一：LLM 推理机制与内部状态诊断

拟定论文题目： 基于隐藏状态几何与置信轨迹的 LLM 推理行为诊断与认知努力量化方法

论文核心问题： 如何从 LLM 内部隐藏状态的几何结构和逐层置信轨迹中，诊断推理行为（冲突、幻觉、认知努力），并区分不同推理阶段的机制差异。

创新点一：隐藏状态吸引子几何驱动的推理故障诊断

大多数现有方法依赖输出层概率或外部验证器来检测 LLM 的推理故障，如幻觉和知识冲突，但这些方法在模型自信输出时完全失效，无法区分"错误但自信"与"正确但自信"的内部状态差异 41518。少数现有方法开始探索隐藏状态空间中的几何特征，如吸引子盆地和流形异常检测，但通常只关注单一故障模式（如幻觉），且缺乏对参数记忆与上下文记忆之间竞争机制的几何刻画 41519。为此，本文提出基于隐藏状态吸引子几何的推理故障诊断框架，通过分析生成过程中隐藏状态向吸引子盆地的收敛轨迹，区分冲突（盆地竞争）与幻觉（盆地缺失）两种故障模式，并利用概率电路对残差流进行密度估计，实现无需外部验证器的实时故障定位。

  • 创新类型： 问题建模创新 + 方法设计创新
  • 创新性判断： 文献池中 4 提出了吸引子几何的理论框架，但仅在合成任务上验证；15 使用概率电路检测幻觉，但未与吸引子几何理论结合。本文创新在于将吸引子几何理论与概率电路密度估计统一为诊断框架，并扩展至冲突与幻觉的联合诊断。

创新点二：基于置信轨迹的认知努力量化与阶段感知分析

大多数现有方法将 LLM 推理视为均匀过程，使用整体指标（如困惑度、准确率）评估性能，忽略了不同推理阶段（如预填充与解码、简单与复杂任务）在认知努力上的差异 11930。少数现有方法开始关注逐层置信轨迹和认知努力量化，但通常将认知努力视为单一标量，未区分不同推理阶段和决策点的贡献差异 192730。为此，本文提出阶段感知的认知努力量化方法，基于逐层置信轨迹的散度模式，分别量化预填充阶段和解码阶段的认知努力，并识别高熵决策点（即 RL 稀疏策略选择的关键位置），以揭示不同推理阶段的计算需求差异。

  • 创新类型： 指标体系创新 + 方法设计创新
  • 创新性判断： 文献池中 1 发现 RL 仅影响 1-3% 的高熵决策点，19 提出认知努力指标但未区分阶段。本文创新在于将认知努力量化与阶段感知分析结合，并利用高熵决策点识别实现细粒度诊断。

论文方向二：模型表示的可解释性与安全评估

拟定论文题目： 面向 LLM 内部表示的多轴安全评估与可解释性归因框架

论文核心问题： 如何系统评估 LLM 内部表示的安全性（包括鲁棒性、公平性、OOD 泛化），并提供可解释的归因分析以定位安全风险来源。

创新点三：基于粒度轴与几何结构的表示安全评估

大多数现有方法对 LLM 安全性的评估集中在输出层行为（如有害内容检测、偏见测试），缺乏对内部表示空间安全属性的系统分析，难以定位安全风险的表示层面根源 91318。少数现有方法开始构建多轴安全评估基准，如 GRL-Safety，但主要针对图表示学习，未覆盖 LLM 内部表示中社会角色粒度、知识记忆几何等关键维度 132528。为此，本文提出基于粒度轴与几何结构的 LLM 内部表示安全评估框架，将社会角色粒度轴（从微观个体到宏观机构）与隐藏状态吸引子几何结合，构建覆盖鲁棒性、公平性、OOD 泛化和解释性的多轴评估体系，并利用粒度轴的主成分对齐特性（余弦 0.972）作为表示健康度的代理指标。

  • 创新类型： 指标体系创新 + 评价方法创新
  • 创新性判断： 文献池中 9 发现粒度轴是角色表示空间的主轴，13 提出 GRL-Safety 多轴评估但针对图模型。本文创新在于将粒度轴作为 LLM 表示安全性的几何代理指标，并构建覆盖社会角色维度的多轴评估体系。

创新点四：元博弈驱动的可解释性归因与安全风险定位

大多数现有归因方法提供一阶特征重要性（如注意力权重、梯度归因），无法捕捉特征之间的交互效应，导致安全风险来源定位不准确 202529。少数现有方法开始探索二阶交互效应，如元博弈框架，但主要应用于通用可解释性任务，未针对安全评估场景进行优化，且计算开销较大 252630。为此，本文提出面向安全评估的元博弈归因方法，将归因方法本身视为合作博弈，计算元归因以量化特征对归因结果的影响，并引入稀疏化策略（基于 28 的动态 Top-K 选择）降低计算开销，实现安全风险来源的精确归因。

  • 创新类型： 方法设计创新 + 泛化迁移创新
  • 创新性判断： 文献池中 25 提出元博弈框架但未针对安全场景优化，28 提出动态稀疏选择但用于自编码器。本文创新在于将元博弈归因与动态稀疏化结合，并专门针对安全评估场景设计归因目标。

论文方向三：高效推理与自适应控制框架

拟定论文题目： 基于预算条件计算与闭环反馈的高效 LLM 推理自适应控制框架

论文核心问题： 如何在保证推理质量的前提下，通过预算条件计算分配和闭环反馈机制，实现 LLM 推理开销的自适应控制。

创新点五：预算条件注意力分配与 MoE 生态控制参数联合优化

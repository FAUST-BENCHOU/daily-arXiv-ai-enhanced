三篇论文方向与六个创新点

论文方向一：LLM 推理效率与 KV Cache 优化

拟定论文题目：面向长上下文推理与 RL 后训练的 KV Cache 压缩与淘汰策略研究

论文核心问题：如何在大语言模型长上下文推理和强化学习后训练场景中，兼顾 KV Cache 的显存压缩效率与模型输出保真度，避免压缩引入的偏差在优化过程中被放大。

创新点一：输出感知的全局 KV Cache 淘汰策略

大多数现有方法基于局部注意力权重进行逐头 KV Cache 淘汰，忽略了值表示、输出投影和跨头交互对 token 贡献度的影响，导致全局重要性排序失真 81317。少数现有方法开始关注 token 贡献度的全局可比性，但尚未将注意力图与投影值状态的乘法交互显式建模为统一的矩阵乘法近似问题，难以实现模型级别的统一选择 813。为此，本文提出 LaProx 策略，将 KV Cache 淘汰重新表述为输出感知的逐层矩阵乘法近似问题，显式建模注意力图与投影值状态的交互以精确量化 token 贡献，并引入全局可比重要性分数实现模型级统一淘汰 81317。

  • 创新类型：方法设计创新
  • 创新性判断：现有方法 8 已提出输出感知的视角，但本文将其与全局可比性分数结合，并引入跨头依赖建模，形成更完整的淘汰框架。文献池中 13 从范围搜索角度处理稀疏注意力，17 从率失真理论处理混合精度量化，均未覆盖输出感知的全局淘汰。

创新点二：RL 后训练中 KV Cache 压缩的偏差校正机制

大多数现有 KV Cache 压缩方法在标准推理场景中近乎无损，但在 RL 后训练中，即使微小的近似误差也会被优化过程的不稳定性急剧放大，导致采样器与学习器之间的策略偏差 717。少数现有方法尝试通过重要性重加权等统计手段缓解偏差，但无法从根本上解决稀疏上下文采样与稠密上下文更新之间的结构不匹配问题 7。为此，本文提出 Shadow Mask Distillation 方法，在 RL 后训练的 rollout 阶段应用 KV Cache 压缩，同时通过影子掩码蒸馏将压缩后的稀疏上下文信息与完整稠密上下文对齐，以消除 off-policy 偏差 71710。

  • 创新类型：方法设计创新 + 约束处理创新
  • 创新性判断：文献池中 7 明确指出了 RL 后训练中 KV Cache 压缩的偏差放大问题，但未提出完整的校正框架。10 提出的梯度外推策略关注的是策略更新方向，而非压缩偏差校正。本文创新点填补了这一空白。

论文方向二：智能体系统可靠性、记忆与运行时治理

拟定论文题目：面向生产环境的 LLM 智能体记忆级联修复与运行时约束治理框架

论文核心问题：如何确保 LLM 智能体在工具调用、记忆级联更新与生产环境部署中的行为可靠性，避免因记忆污染、信号反转或约束缺失导致的级联失效。

创新点三：屏障优先的智能体记忆级联修复契约

大多数现有智能体记忆系统仅关注单次记忆的读写正确性，忽略了当源工件被删除、修正或工具迁移导致失效时，派生工件仍可见并引导后续行为的问题，即级联更新问题 51811。少数现有方法开始关注记忆的版本管理或一致性维护，但缺乏形式化的修复契约来保证派生状态在修复事件中的可控转换 518。为此，本文提出 MemoRepair 的屏障优先级联修复契约，通过撤回受影响派生、从保留支撑和已修复前驱构建后继、以及限制发布为已验证的前驱闭包后继，实现派生状态到验证状态的可控转换 51811。

  • 创新类型：问题建模创新 + 方法设计创新
  • 创新性判断：文献池中 5 已形式化级联更新问题并提出 MemoRepair 契约，但本文将其与 18 的规模条件评估协议结合，引入可用的规模边界概念，形成更完整的修复决策框架。11 的 WiCER 方法关注的是知识编译而非记忆修复，属于不同问题域。

创新点四：运行时约束优先的智能体治理架构

大多数现有智能体治理控制附着于提示词、仪表盘或事后文档，导致在受监管场景中，本应约束执行过程的义务往往在执行完成后才被评估，形成结构性不匹配 281224。少数现有方法开始将约束嵌入运行时架构，但通常依赖有限奖励惩罚来替代硬约束，无法保证规范-轨迹对应关系的最小不变量 28。为此，本文提出 SARC 运行时治理架构，将约束作为与状态、动作空间和奖励并列的一等规范对象，通过预动作门、动作时监控器、事后审计器和升级路由器四个执行点实施约束，并形式化规范-轨迹对应所需的最小不变量 281224。

  • 创新类型：系统框架创新 + 约束处理创新
  • 创新性判断：文献池中 28 已提出 SARC 架构，但本文将其与 12 的架构-生命周期框架和 24 的信号反转分析结合，引入方向感知的自适应学习机制，使约束执行能够根据信号方向动态调整。24 揭示的“相同信号、相反含义”现象进一步强化了硬约束而非软惩罚的必要性。

论文方向三：异构工作流调度与资源协同

拟定论文题目：面向 LLM 多阶段工作流与 AI-RAN 融合基础设施的未来状态感知调度与分层资源分配

创新点五：未来状态感知的异构 LLM 工作流调度
